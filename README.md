## 実行環境
Python 3.6.9

NTT コミュニケーションズが開発の、日本語辞書を活用した自然言語処理、音声認識 API プラットフォーム [COTOHA API](https://api.ce-cotoha.com/) を用いています. 

### 必要なライブラリ
- jaconv
- kajize
